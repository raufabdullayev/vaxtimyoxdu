Guclu Terefleri

  • Minimum asillilik - Cemi 10 production dependency. Coh yungul ve suretli layihe.
  • 100% statik generasiya (SSG) - Butun 30 sehife build zamani yaradilir. Vercel CDN-den suresli xidmet olunur.
  • Agilli AI arxitekturasi - Groq (pulsuz) → Gemini (pulsuz) → OpenAI (odenisli) fallback zenciri ile xercler minimuma endirilir.
  • Mexfilik ustunluyu - 11/14 alet tamamen brauzerde isleyir. Server-e hec bir istifadeci melumati gonderilmir.
  • Guclu SEO bazasi - JSON-LD structured data, sitemap, robots.txt, canonical URL, OG teqleri - hamisi var.
  • Kod ayirma (Code Splitting) - Her alet next/dynamic ile ayrica yuklenir. Istifadeciler yalniz istifade etdikleri aletin JS-ini yukleyir.
  • Tehlukesizlik basliklari - X-Content-Type-Options, X-Frame-Options, Referrer-Policy, Permissions-Policy konfiqurasiya edilib.
  • Duzgun Server/Client komponent serhedi - Yalniz interaktiv komponentler 'use client' istifade edir. Layout/Footer Server Component olaraq qalir.
  • Temiz cachleme strategiyasi - Statik fayllar 1 il immutable cache, seqiller 30 gun stale-while-revalidate.
  • Konsistent dizayn sistemi - HSL deyisenler uzerinde qurulan token-li renk sistemi (shadcn/ui patterni).

Kritik Problemler

🚨 DERHAL DIQQET TELEB EDEN MESELELER Asagidaki problemler produksiyaya cixmazdan evvel mutleq hell edilmelidir.

6.1 Tehlukesizlik Problemleri

# Problem Ciddilik Hell vaxti
S-1 Next.js 14.2.21-de 10 CVE var (icaze bypass daxil). 14.2.35-e yenilemek lazim. KRITIK 15 deq
S-2 Groq API key deyisdirilmelidir - key artiq bu audit zamani expose olub. KRITIK 10 deq
S-3 CORS konfiqurasiyasi yoxdur - istenilen sayt API endpoint-lerinizi istifade ede biler. YUKSEK 30 deq
S-4 Content-Security-Policy (CSP) yoxdur - XSS hucumuna aciq. YUKSEK 45 deq
S-5 CSRF qorumasi yoxdur POST API endpoint-lerinde. YUKSEK 1 saat

6.2 Arxitektura Problemleri

# Problem Ciddilik
A-1 Rate limiter serverless-de islemir. In-memory Map her cold start-da sifirlanir. Vercel-de her funksiya ferqli konteyner-de ishleye biler - rate limit kecirilir. AI API key-ler qorumasizdir. KRITIK
A-2 Kontent hardcode edilib. Blog yazilar ve xeber meqaleleri birbase TSX fayllarin icindedir. Yeni kontent elave etmek ucun kod deyishikliyi ve yeniden deploy lazimdir. YUKSEK
A-3 Brend uygunsuzlugu. About, Privacy, Terms sehifelerinde "ToolBox AI" ve "toolbox-ai.com" yazilir. Layihenin adi "Vaxtim Yoxdu" olmalidir. YUKSEK
A-4 AI endpoint-lerde timeout yoxdur. Provider asildiqda funksiya hemislik gozleyir (Vercel-in oz timeout-u olene qeder). YUKSEK
A-5 toolbox-ai dublikat kodbazadir. Eyni 14 alet, eyni API routes. Iki ayri repo saxlamaq texniki borc yaradir. ORTA

6.3 SEO Problemleri

# Problem Ciddilik
SEO-1 hreflang pozuqdur. 'az' ve 'en' eyni URL-e isare edir. Bu semantik olaraq menasizdir ve Google-u cesdire biler. KRITIK
SEO-2 <html lang="az"> hardcode edilib amma blog ve tools sehifeleri ingilis dilindedir. Ekran oxuculari ve axtaris motorlari ucun zedekan. KRITIK
SEO-3 Butun 30 sehife ucun cemi 1 OG sekli var. Sosial media paylasiminda her sehife eyni goruntunu gosterir. CTR-i dramatik azaldir. YUKSEK
SEO-4 Kontent coh nazikdir. Xeberler ~100 soz, bloglar ~200 soz. Google Helpful Content telimatina gore bunlar "thin content" hesab olunur. YUKSEK
SEO-5 Favicon / app icon yoxdur. Brauzer tabinda hec ne gorunmur. web manifest de yoxdur. YUKSEK
SEO-6 Blog ve alet sehifeleri arasinda carpaz link yoxdur. Boyuk SEO fuqseti itirilir. YUKSEK
SEO-7 Sitemap lastModified her build-de new Date() istifade edir. Google ucun faktiki kontent tarixini gostermek lazimdir. ORTA

6.4 UI/UX Problemleri

# Problem Ciddilik
UI-1 Focus indikatorlari yoxdur. Klaviatura istifadecileri interaktiv elementlerin harada olduqunu gore bilmir. KRITIK
UI-2 Dark mode konfiqurasiya edilib amma toggle yoxdur. CSS deyisenleri, Tailwind config - hamisi hazirdir, amma istifadeci aktivleshdire bilmir. Olu koddur. YUKSEK
UI-3 Alet sehifelerinde ikiqat reklam. Sehife seviyyesinde 2 + Template seviyyesinde 2 + Sidebar-da 2 = 6 reklam her alet sehifesinde. AdSense siyasetini poza biler. YUKSEK
UI-4 Fayl yukleme zonalari klaviatura ile elcatmaz. PDF Merge, Image Compress, Image Convert aletlerinde div onClick istifade edilir - keyboard users ucun islemir. YUKSEK
UI-5 @tailwindcss/typography plugin yoxdur. Blog/xeber sehifelerinde istifade olunan 'prose' class-lari hec bir effekt vermir. YUKSEK
UI-6 Custom 404, error, loading sehifeleri yoxdur. Next.js default sehifelerine etibar edilir. ORTA

Tehlukesizlik Auditi (Detallar)

Movcud Qorumalar (Yaxsi isler)

  • .gitignore duzgun konfigurasiya edilib - .env.local, .vercel, *.pem hamisi ignore olunur
  • Server-terefli input validasiyasi - butun 3 API route input tipini, movcudlugunu ve uzunlugunu yoxlayir
  • Generik xeta mesajlari - API route-lar "Service temporarily unavailable" qaytarir, daxili detallari aciqlmir
  • poweredByHeader: false - server finqerprintini gizledir
  • X-Frame-Options: DENY - clickjacking-in qarsisini alir
  • Permissions-Policy - camera, microphone, geolocation mehdudlasdirilir
  • Ucuncu teref skriptler tehlukesiz yuklenir - GA4 afterInteractive strategiyasi ile
  • dangerouslySetInnerHTML istifadeci inputu ile istifade olunmur - yalniz statik JSON-LD ucun
  • AI cavablari metn olaraq render edilir (React auto-escape) - XSS riski yoxdur
  • Hec bir eval() ve ya Function() - dinamik kod icraati yoxdur

Reklam Yerleshmesi Problemi

🚨 Alet sehifelerinde 6 reklam! Her alet sehifesinde: 2 reklam ustde (sehife + template), 2 reklam altda (sehife + template), 2 reklam sidebar-da. Bu AdSense siyasetini poza biler ve istifadeci tecrubsini ciddi zeifletir.

Accessibility Catismazliqlari

  • Skip-to-content linki yoxdur - klaviatura istifadecileri her sehifede header-den kecmelidir
  • Focus indikatorlari yoxdur - elemenlerin cogunluqunda focus gorsenliyi yoxdur

  • Semantik HTML elementleri istifade olunur (header, main, footer, nav, article, aside)
  • Butun textarea-larda label elementleri var
  • Mobil menyu toggle-da aria-label="Menyu" var
