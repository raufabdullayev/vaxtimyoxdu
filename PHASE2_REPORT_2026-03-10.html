⚠ Pipeline Problemleri ve Helleri

Problem 1: YAML Syntax Xetasi Duzeldildi
GitLab CI-da only: (kohne syntax) ve rules: (yeni syntax) eyni pipeline-da istifade olunurdu. GitLab bunu desteklemir ve pipeline-i dərhal reject edir.

Evvel (Xetali)

validate: only: [main, merge_requests]
preview: rules: [{if: ...}]

Sonra (Duzeldilmis)

validate: rules: [{if: '$CI_COMMIT_BRANCH == "main"'}]
preview: rules: [{if: ...}]
Problem 2: Shell Substitution Duzeldildi
Preview job-da PREVIEW_URL=$(vercel deploy ...) GitLab CI parser terefinden reject olunurdu. pipe tee ile evez edildi.
Problem 3: Node.js Versiya Uygunsuzlugu Duzeldildi
Vercel project settings-de Node.js 24.x qoyulmusdur, amma Vercel maksimum 22.x destekleyir. Vercel API vasitesile 20.x-e duzeldildi.
